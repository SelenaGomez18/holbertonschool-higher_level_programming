The ``say_my_name`` module
==========================

Using ``say_my_name``
---------------------

First import ``say_my_name`` from the ``3-say_my_name`` module:

    >>> say_my_name = __import__('3-say_my_name').say_my_name

Now use it:

    >>> say_my_name("Juan", "Pérez")
    My name is Juan Pérez

    >>> say_my_name("María")
    My name is María

    >>> say_my_name("", "López")
    My name is  López

    >>> say_my_name("", "")
    My name is
